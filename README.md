# LingMate

LingMate is an AI-powered language learning platform focused on conversational practice and scenario-based learning.

---

## 🚀 Features

- 💬 Free Talk mode — natural AI conversation  
- 🎯 Scenario-based lessons (real-life situations)  
- 📈 Adaptive vocabulary tracking  
- 🎤 Voice input (browser)  

---

## 🧠 Concept

LingMate combines:
- conversational AI (LLM)
- structured lessons
- personalized learning experience

The system adapts to the user level and supports continuous dialogue.

---

## 🛠 Tech Stack

- Python (FastAPI)
- Gemini API (LLM)
- HTML / CSS / JavaScript
- Web Speech API

---

## ⚙️ Setup

1. Install dependencies:

```bash
pip install -r requirements.txt
```

2. Create a `.env` file based on `.env.example`

3. Add your API key to `.env`

4. Run the project:

```bash
python main.py
```
